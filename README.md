# File Storage Website

A lightweight static web application designed to provide a simple and efficient way to organize and share documents.

---

## Live Demo

https://vhlyen.github.io/file-storage/

---

## Project Motivation

Accessing and sharing learning materials is often fragmented across different platforms.
This project was built to create a centralized, minimal interface where documents can be organized and distributed in a clear and accessible manner.

---

## Solution

The application uses a static web architecture to deliver a fast and straightforward user experience.
Users can browse available files, understand their context through descriptions, and download them directly without additional steps such as authentication or navigation overhead.

---

## Key Features

* Structured display of shared documents
* Contextual descriptions for better understanding of each resource
* Direct file download through a clean interface
* Fully static deployment for performance and simplicity

---

## Technical Approach

The project is implemented using HTML and CSS, focusing on layout clarity and responsive design.
It is deployed via GitHub Pages, enabling fast and reliable access without backend complexity.

---

## Future Development

This project is intended to evolve into a more complete document-sharing platform:

* Search and filtering system for better discoverability
* File preview functionality (e.g. PDF, images)
* Upload and content management features
* Transition toward a dynamic architecture using cloud services
